The 1000 Genomes project is holding a tutorial during ASHG 2014.

The 1000 Genomes Project has released the variants, genotypes, and integrated haplotypes for the complete set of 2504 samples from 26 populations.  This tutorial describes the data sets, how to access them, and how to use them.

The tutorial will be on Sunday 19th between 8 and 9:30pm in the Convention Centre, Room 24ABC in the Upper Level.

The program for the tutorial is

8:00 - 8:12 Phase 3 release description – Shane A. McCarthy
8:12 - 8:22 How to access the data – Laura Clarke
8:22 - 8:34 Structural variants – Eugene Gardner
8:34 - 8:46 Population genetics – Ryan Hernandez
8:46 - 8:58 Y chromosome – Yali Xue
8:58 - 9:08 Imputation server - Christian Fuchsberger
9:08 - 9:20 Using the 1000 Genomes data for disease studies – Karen L. Mohlke
9:20 - 9:30 Q&A

No registration is needed for this meeting.

Slides from Goncalo Abecasis's Plenary presentation about the completion of the project are also available.

Laura Clarke's poster about 1000 Genomes Data Access is also available.

The complete slide set and where available powerpoint format slides can be found on our FTP site in technical/working/20141031_ASHG14_tutorial/

Please send any questions about this meeting to info@1000genomes.org
