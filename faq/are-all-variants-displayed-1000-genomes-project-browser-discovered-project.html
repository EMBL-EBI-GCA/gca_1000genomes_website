Are all the variants displayed on the 1000 Genomes Project Browser discovered by the project?

  • browser
  • data
  • dbSNP
  • Ensembl
  • source
  • variants
Answer: 

No, not all the variants in the 1000 Genomes Browser were discovered by the 1000 Genomes Project.  

The content of the 1000 Genomes Project Browser is based on a custom version of the Ensembl browser. These databases contain the Ensembl core features (genes and transcripts), regulatory elements from the Ensembl Regulatory Build and variation data from Ensembl Variation database.

We load the newest version of the 1000 genoems data if it is not yet in dbSNP. As well as our variation data, Ensembl variation contains data from dbSNP, ClinVar, COSMIC, dbGaP, dbVAR, EGA and many other sources. We currently use the Ensembl 75 databases. For a complete list of the sources for the Ensembl Variation, please see http://feb2014.archive.ensembl.org/info/genome/variation/sources_documentation.html for details.
