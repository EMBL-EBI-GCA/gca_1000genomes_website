Can I get haplotype data for the 1000 genomes individuals

  • Data Analysis
  • genotypes
  • imputation
  • vcf
Answer: 

Our current most recent release, the Phase1 integrated data set, contains fully phased haplotypes for 1092 individuals using both low coverage and exome data and has snps, indels and larger deletions. Please note the indels and larger deletions were modelled as point events so there may be some inconsistencies in the haplotypes which surround those events.
