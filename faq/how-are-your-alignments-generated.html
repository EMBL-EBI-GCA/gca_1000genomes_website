How are your alignments generated?

  • bam
  • Data Analysis
  • tools
Answer: 

The 1000 genomes project has used several different alignment algorithms during its duration. For the pilot project we used MAQ for Illumina, Corona lite for SOLiD and ssaha for 454. For the main project we use BWA for Illumina and BFAST for SOLiD, 454 started with ssaha but the final set of 454 alignments used smalt. The full process is described in the README
