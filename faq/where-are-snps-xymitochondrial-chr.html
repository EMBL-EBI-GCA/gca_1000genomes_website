Where are the snps for the X/Y/Mitochondrial chr

  • Data Access
  • variants
Answer: 

Chromosome X, Y and MT variants are available for the phase 1 variant set. The chrX calls were made in the same manner as the autosome variant calls and as such are part of the integrated call set and include snps, indels and large deletions, note the males in this call set are show as haploid for any chrX snp not in the Pseudo Autosomal Region (PAR). The chrY and MT calls where made independently. We have snps and deletions for chrY and snp calls for chrMT. For more details about how these call sets were made please see the phase1 paper.
