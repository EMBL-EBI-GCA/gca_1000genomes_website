Is there any functional annotation for the 1000 Genomes phase 1 data?

  • Data Access
  • Data Analysis
  • functional annotation
  • Phase 1
Answer: 

As part of our phase1 analysis we performed functional annotation of our phase1 variants with respect to both coding and non coding annotation from Gencode and the Encode project respectively.

This functional annotation can be found in our phase1 analysis results directory. We present both the annotation we compared the variants too and vcf files which contain the functional consequences for each variant.
