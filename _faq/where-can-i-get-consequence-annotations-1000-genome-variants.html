The 1000 genomes phase1 analysis calculated consequences based on Gencode annotation and this can be found in the directory phase1/analysis_results/functional_annotation/

Ensembl can also provides consequence information for the variants. The variants that are loaded into the Ensembl database and have consequence types assigned and displayed on the Variation view. Ensembl can also offer consequence predictions using their Variant effect predictor.

Please note the phase1 annotations and the Ensembl annotations visible via the browser due to using different versions of gene and non coding annotation.
