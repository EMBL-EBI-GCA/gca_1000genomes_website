The majority of our official release vcf files contain phased variants. This is also true for both our pilot and phase1 data sets. You can tell when a VCF file contains a phased genotype as the delimited used in the GT field is a pipe symbol | e.g

#CHROM  POS     ID      REF     ALT     QUAL    FILTER  INFO    FORMAT  HG00096
10   60523  rs148087467    T     G       100     PASS    AC=0;AF=0.01;AFR_AF=0.06;AMR_AF=0.0028;AN=2; GT:GL 0|0:-0.19,-0.46,-2.28

The phase1 release file do contain global R2 values but you can also use our vcf to plink converter if you wish to use our files with haploview or another similar tool.
